# Angular + Rails example

Originally made this as a demo for [Rails + Angular equation](http://dev.mikamai.com/post/97732414859/the-rails-angular-equation-can-give-you-satisfaction)

## Setup

    bundle install
    rake bower:install
    rake db:migrate db:seed
